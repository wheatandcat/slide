# コード

以下のようなコードで認証は実装可能[^1]

```ts {all|1,5|14|all}
  const handleOAuthLogin = async (provider: Provider, scopes = "email") => {
    await supabase.auth.signInWithOAuth({
      provider,
      options: {
        scopes,
        redirectTo: location.href
      }
    })
  }

  略）
```
